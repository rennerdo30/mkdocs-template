# Changelog

All notable changes to this project are documented here.

---

## [1.0.0] - 2024-01-15

### Added

- Initial release of the Luminous Void theme
- Dark mode with warm coral and electric cyan accents
- Custom typography with Satoshi and Inter fonts
- Fira Code for code blocks with VS Code Dark+ highlighting
- Glassmorphic UI elements
- Grain texture background
- Smooth animations and transitions
- Custom admonition styling
- Feature cards for homepage
- Hero section component
- Responsive design for all devices

### Documentation

- Complete getting started guide
- Configuration reference
- Feature showcase
- API documentation examples
- Best practices guide

---

## [0.2.0] - 2024-01-10

### Added

- Content tabs support
- Mermaid diagram integration
- Keyboard key styling
- Custom scrollbar design

### Changed

- Improved code block styling
- Enhanced table appearance
- Better navigation hover states

### Fixed

- Mobile navigation overflow
- Search result highlighting
- Footer link alignment

### Added

This project follows [Semantic Versioning](https://semver.org/):

- **MINOR**: New features, backwards compatible
- **PATCH**: Bug fixes, backwards compatible
